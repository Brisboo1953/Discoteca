DiscoTK es una página web para la venta de discos y vinilos de artistas de todo el mundo a precios accesibles. El sitio muestra un catálogo de discos cargados desde un archivo 

Cuenta con un sistema de inicio y cierre de sesión simulado, donde los favoritos del usuario se guardan en el localStorage. Los usuarios pueden marcar discos como favoritos, agregarlos a un carrito de compras y ver el total de su compra.
